# Personal Site
A static site built with Eleventy, Tailwind CSS, and Bun.

## Setup
1. Install dependencies: `bun install`
2. Run dev server: `bun run start`
3. Build for production: `bun run build:prod`
4. Clean output: `bun run clean`
